# JobVerse | Elite Career Hub

![Banner](https://images.unsplash.com/photo-1497215728101-856f4ea42174?auto=format&fit=crop&q=80&w=1200)

JobVerse is a premium, high-fidelity job board application engineered for the modern workforce. Built with a powerhouse stack (Django & React) and a refined aesthetic, it offers a seamless interface for both top-tier talent and innovative engineering teams.

## 🚀 Key Features

*   **Dynamic Intelligence Grid**: High-performance job searching with real-time filtering.
*   **Premium Brand Profiles**: Visual-first company directories featuring brand cover photos and verified logos.
*   **Role-Based Access Control**: Tailored experiences for both Seekers and Employers.
*   **Advanced UI/UX**: Powered by Framer Motion, Tailwind CSS, and Lucide icons for a senior-developer standard.
*   **Django-Powered Backend**: Robust API infrastructure for secure data management and authentication.

## 🛠 Tech Stack

*   **Frontend**: React (Vite), TypeScript, Framer Motion, Tailwind CSS, Lucide Icons.
*   **Backend**: Python (Django), Django REST Framework, JWT Authentication.
*   **Database**: PostgreSQL / SQLite (Development).

## 📦 Installation & Setup

### Prerequisites
- Node.js (v18+)
- Python (v3.10+)

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
